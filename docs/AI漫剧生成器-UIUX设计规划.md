# AI 漫剧创作台 UI/UX 设计规划 V2.0

> 版本：V2.0  
> 日期：2026-06-29  
> 目标：在 V1.0 基础上，根据整合版建议优化用户路径、导航方式和编辑体验

---

## 一、设计定位

### 1.1 主题与氛围
产品主题是 **"AI 漫剧创作"**——将文字剧情、角色、镜头转化为连续的画面与视频。设计应传递出：

- **想象力**：像翻开一本未完成的漫画分镜册
- **剧场感**：暗色观影氛围，突出画面内容
- **专业感**：镜头、分镜、时间线等影视创作工具隐喻
- **科技感**：AI 生成、模型选择、参数调节等智能工具属性
- **亲和力**：不要让新手感到复杂和压迫

### 1.2 设计语言关键词
`剧场` `分镜` `胶片` `漫画网点` `暗房` `导演台` `故事板` `放映机` `霓虹弱光` `草稿纸`

### 1.3 避免的模板化方向
- ❌ 奶油色背景 + 高对比衬线 + 陶土色点缀
- ❌ 纯黑背景 + 单一荧光绿/朱红点缀
- ❌ 报纸风密集栏线 + 零圆角
- ❌ 过度复杂的仪表盘式布局

我们要做的是：**暗色影院感 + 漫画分镜网格 + 编辑区适度提亮**，让观影体验和创作体验都舒适。

## 二、设计系统 V2.0

### 2.1 配色方案

| 名称 | Hex | 用途 |
|------|-----|------|
| 影院黑 | `#0B0C0F` | 视频预览区、放映模式背景 |
| 深灰板 | `#14161B` | 面板、卡片、侧边栏背景 |
| 中灰板 | `#1E2128` | 输入框、编辑区背景 |
| 编辑灰 | `#242830` | 文本编辑区专用背景，比中灰板稍亮 |
| 分割线 | `#2A2E36` | 边框、分隔线、网格线 |
| 漫剧紫 | `#7C5CFF` | 主品牌色，AI 与想象力 |
| 荧光青 | `#00E5C9` | 高亮、进度、成功、选中状态 |
| 暖光橙 | `#FF9E5E` | 警告、重点、情绪温度 |
| 纯文本白 | `#F2F3F5` | 主要文字 |
| 次级文本 | `#9AA0AC` | 辅助说明、占位符 |
| 禁用灰 | `#5A6070` | 禁用状态 |

### 2.2 编辑区亮度调整（V2.0 新增）

为避免长时间文字编辑的视觉疲劳：
- 故事输入框、提示词编辑区、字幕编辑区使用 `#242830` 编辑灰背景
- 这些区域的外边框使用 `#3A3F48`，比主分割线更柔和
- 视频预览区、项目面板保持深灰板

### 2.3 字体方案

| 用途 | 字体 | 说明 |
|------|------|------|
| 标题/Display | `Space Grotesk` | 几何感、现代、科技气质 |
| 正文/UI | `Inter` | 清晰、中性、屏显优化 |
| 数据/代码 | `JetBrains Mono` | API Key、参数值、时长等 |
| 中文正文 | `Noto Sans SC` | 中文可读性 |

| Token | 值 | 用途 |

## 三、标志性设计元素

### 核心签名元素：**"分镜网格 + 动态镜头框"**

- 首页 Hero 用 2×3 分镜网格展示示例漫剧画面
- 分镜编辑页本身就是网格
- 选中分镜时，镜头框带有青色 glow 脉冲动画
- 生成进度以"胶片齿孔"式进度条呈现
- 新增：故事板阅读模式，像翻漫画书一样浏览分镜

## 四、核心页面结构 V2.0

### 4.1 页面总览（从 8+ 页收敛为 6 页）

| 页面 | 路径 | 说明 |
|------|------|------|
| 首页 | `/` | 产品介绍、示例、开始创作 |
| 项目列表 | `/projects` | 最近创作、草稿、成片 |
| 创作工作台 | `/projects/:id/studio` | 角色、故事、分镜、预览的主工作区 |
| 生成队列 | `/projects/:id/generate` | 视频任务、重试、结果管理 |
| 后期导出 | `/projects/:id/export` | 预览、字幕、音频、导出 |
| 模型中心 | `/settings/models` | 模型商店 + Key 管理 |

### 4.2 首页

保持 V1.0 设计：
- 暗色影院背景 + 紫色渐变光晕
- 分镜网格作为 Hero 视觉中心
- 功能特性区、创作流程区、CTA

变化：
- 主 CTA 改为"立即创作第一条漫剧"，强调极速模式
- 次 CTA 为"配置模型"，面向有准备的用户

### 4.3 项目列表

保持 V1.0 卡片设计：
- 项目卡片展示状态、进度、分镜数
- 渐变进度条 + hover glow
- "新建项目" 虚线卡片

变化：
- 增加"最近创作"标签
- 支持极速模式入口："30秒开始新创作"

## 七、响应式策略

- **桌面端（1280px+）**：完整双栏/三栏布局
- **平板端（768px-1279px）**：侧边栏收起为图标，主内容区全宽
- **移动端（<768px）**：仅保留轻量功能，复杂编辑引导到桌面端

创作流程中的分镜编辑、时间线等复杂操作**优先保证桌面体验**。

---

## 八、设计原则总结

1. 暗色影院背景用于视频预览，编辑区适度提亮
2. 分镜网格作为标志性视觉元素
3. 紫色 + 青色作为 AI 与镜头的功能色
4. Space Grotesk + Inter + JetBrains Mono 字体组合
5. 从用户视角写文案，避免技术术语
6. 动效克制，只在关键交互处使用
7. 检查表式导航替代线性向导
8. 模型商店与 Key 配置在同一视图完成
9. 组件化设计，保证各页面体验一致
