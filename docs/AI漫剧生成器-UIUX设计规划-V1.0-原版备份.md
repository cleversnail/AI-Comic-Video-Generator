# AI 漫剧生成器 UI/UX 设计规划

> 版本：V1.0  
> 日期：2026-06-29  
> 目标：为 AI 漫剧生成器（PC Web）建立独特、有主题感的视觉设计系统，并规划关键页面结构

---

## 一、设计定位

### 1.1 主题与氛围
产品主题是 **"AI 漫剧创作"**——将文字剧情、角色、镜头转化为连续的画面与视频。设计应传递出：

- **想象力**：像翻开一本未完成的漫画分镜册
- **剧场感**：暗色观影氛围，突出画面内容
- **专业感**：镜头、分镜、时间线等影视创作工具隐喻
- **科技感**：AI 生成、模型选择、参数调节等智能工具属性

### 1.2 设计语言关键词
`剧场` `分镜` `胶片` `漫画网点` `暗房` `导演椅` `故事板` `放映机` `霓虹弱光`

### 1.3 避免的模板化方向
- ❌ 奶油色背景 + 高对比衬线 + 陶土色点缀
- ❌ 纯黑背景 + 单一荧光绿/朱红点缀
- ❌ 报纸风密集栏线 + 零圆角

我们要做的是：**暗色影院感 + 漫画分镜网格 + 可控的参数面板**，既有创作工具的秩序，又有漫剧的叙事氛围。

## 二、设计系统

### 2.1 配色方案

| 名称 | Hex | 用途 |
|------|-----|------|
| 影院黑 | `#0B0C0F` | 主背景，模拟观影环境 |
| 深灰板 | `#14161B` | 面板、卡片、侧边栏背景 |
| 中灰板 | `#1E2128` | 输入框、hover 状态、次级面板 |
| 分割线 | `#2A2E36` | 边框、分隔线、网格线 |
| 漫剧紫 | `#7C5CFF` | 主品牌色，代表 AI 与想象力 |
| 荧光青 | `#00E5C9` | 高亮、进度、成功、镜头焦点 |
| 暖光橙 | `#FF9E5E` | 警告、重点、情绪温度 |
| 纯文本白 | `#F2F3F5` | 主要文字 |
| 次级文本 | `#9AA0AC` | 辅助说明、占位符 |
| 禁用灰 | `#5A6070` | 禁用状态 |

### 2.2 字体方案

| 用途 | 字体 | 备选 | 说明 |
|------|------|------|------|
| 标题/Display | `Space Grotesk` | `Inter Display` | 几何感、现代、略带科技气质 |
| 正文/UI | `Inter` | `SF Pro Text` | 清晰、中性、屏显优化 |
| 数据/代码 | `JetBrains Mono` | `SF Mono` | 用于 API Key、参数值、日志 |
| 中文正文 | `Noto Sans SC` | `PingFang SC` | 保证中文可读性 |

### 2.3 字体尺度

| Token | 大小 | 字重 | 用途 |
|-------|------|------|------|
| Display XL | 48px | 700 | 首页大标题 |

| H3 | 18px | 500 | 小标题 |

| Caption | 13px | 400 | 辅助说明 |
| Small | 12px | 500 | 标签、状态 |

### 2.4 间距系统

| Token | 值 | 用途 |
|-------|-----|------|
| xs | 4px | 图标与文字间距 |
| sm | 8px | 紧凑元素间距 |
| md | 16px | 卡片内边距、表单间距 |
| lg | 24px | 模块间距 |
| xl | 32px | 大模块间距 |
| 2xl | 48px | 页面级间距 |
| 3xl | 64px | 首页 section 间距 |

### 2.5 圆角与阴影

| Token | 值 | 用途 |
|-------|-----|------|
| radius-sm | 6px | 小按钮、标签 |
| radius-md | 10px | 输入框、卡片 |
| radius-lg | 16px | 大卡片、弹窗 |
| radius-xl | 24px | 首页 Hero 元素 |

阴影以**内发光/外发光**为主：
- 主按钮 hover：`0 0 20px rgba(124, 92, 255, 0.35)`
- 视频预览框：`0 0 40px rgba(0, 229, 201, 0.08)`
- 卡片选中：`inset 0 0 0 1px rgba(124, 92, 255, 0.5)`

## 三、标志性设计元素

**核心签名元素："分镜网格 + 动态镜头框"**

- 首页 Hero 用 2×2 或 3×3 的分镜网格展示示例漫剧画面
- 分镜编辑页本身就是网格
- 选中分镜时，镜头框带有动态高亮边框（青色 glow）
- 生成进度以"胶片齿孔"式进度条呈现

## 六、动效规划

| 场景 | 动效 |
|------|------|
| 分镜卡片 hover | 轻微上浮 + 边框 glow |
| 选中分镜 | 青色边框脉冲动画 |
| 生成进度 | 胶片齿孔式进度条流动 |
| 视频生成完成 | 卡片缩放入场 |
| 页面切换 | 淡入 + 轻微上滑 |
| 模型验证 | 按钮加载 → 成功对勾 |

## 七、响应式策略

- **桌面端（1280px+）**：完整三栏/双栏布局
- **平板端（768px-1279px）**：侧边栏收起
- **移动端（<768px）**：仅保留轻量功能，复杂编辑引导到桌面端

---

## 八、设计原则总结

1. 深色影院背景，让画面内容成为视觉中心
2. 分镜网格作为标志性视觉元素
3. 紫色 + 青色作为 AI 与镜头的功能色
4. Space Grotesk + Inter + JetBrains Mono 字体组合
5. 从用户视角写文案
6. 动效克制，只在关键交互处使用
7. 组件化设计，保证各页面体验一致
