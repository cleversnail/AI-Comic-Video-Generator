# P1-4 角色锁定强度模块开发说明

## 模块概述

**模块名称**：角色锁定强度（Character Lock Level）  
**优先级**：P1-4  
**预计工时**：1天  
**完成状态**：✅ 已完成

### 功能描述

角色锁定强度是角色一致性系统的核心功能之一，通过三档锁定强度（宽松/中等/严格）控制AI生成时角色形象的一致性程度。

### 锁定强度定义

| 强度 | 适用场景 | 系统行为 |
|------|----------|----------|
| **宽松 (loose)** | 创意发散、不同画风尝试 | 主要注入文字描述，允许造型变化 |
| **中等 (medium)** | 常规漫剧生成 | 使用参考图 + 角色提示词，兼顾一致性和画面自由度 |
| **严格 (strict)** | 连载角色、品牌IP | 强制参考主形象/变体，限制发型、服装、脸部变化 |

### 1. 后端改动

#### 1.1 CharactersService（增强提示词生成策略）

**文件**：`apps/ai-video-api/src/modules/characters/characters.service.ts:166-250`

**改动内容**：
- **重构 `buildCharacterPrompt` 方法**：根据锁定强度动态调整角色描述注入方式
  - `strict`：详细描述 + 强制一致性约束词（如 `consistent character design, identical face features`）
  - `medium`：标准描述 + 适度约束词（如 `consistent character, maintain recognizable features`）
  - `loose`：简化描述，允许创意变化（如 `character design, artistic interpretation allowed`）
- **新增 `getLockLevelInfo` 方法**：获取锁定强度的描述信息（标签、描述、颜色）
- **新增 `updateLockLevel` 方法**：更新角色锁定强度

**关键代码逻辑**：

```typescript
buildCharacterPrompt(character: any, lockLevelOverride?: string): string {
  const lockLevel = lockLevelOverride || character.lockLevel || 'medium';
  const parts: string[] = [];

  if (character.name) parts.push(character.name);
  if (character.gender) parts.push(character.gender);
  if (character.age) parts.push(`${character.age} years old`);

  switch (lockLevel) {
    case 'strict':
      if (character.appearance) parts.push(`detailed appearance: ${character.appearance}`);
      if (character.outfit) parts.push(`exact outfit: ${character.outfit}`);
      parts.push('consistent character design, identical face features, ...');
      break;
    case 'medium':
      if (character.appearance) parts.push(character.appearance);
      if (character.outfit) parts.push(`wearing ${character.outfit}`);
      parts.push('consistent character, maintain recognizable features');
      break;
    case 'loose':
    default:
      if (character.appearance) parts.push(`general appearance: ${character.appearance.substring(0, 50)}...`);
      parts.push('character design, artistic interpretation allowed');
      break;
  }

  return parts.join(', ');
}
```

### 2. 前端改动

#### 2.1 API层（添加角色API调用）

**文件**：`apps/ai-video-web/lib/api.ts:50-157`

**新增类型**：
- `Character`：角色对象类型
- `CreateCharacterDto`：创建角色DTO
- `LockLevelInfo`：锁定强度信息

**新增API方法**：

```typescript
export const charactersApi = {
  listCharacters: (projectId: string) => api.get<{ data: Character[] }>(`/projects/${projectId}/characters`).then((r) => r.data.data),
  getCharacter: (projectId: string, characterId: string) => api.get<{ data: Character }>(`/projects/${projectId}/characters/${characterId}`).then((r) => r.data.data),
  createCharacter: (projectId: string, data: CreateCharacterDto) => api.post<{ data: Character }>(`/projects/${projectId}/characters`, data).then((r) => r.data.data),
  updateCharacter: (projectId: string, characterId: string, data: Partial<CreateCharacterDto>) => api.put<{ data: Character }>(`/projects/${projectId}/characters/${characterId}`, data).then((r) => r.data.data),
  deleteCharacter: (projectId: string, characterId: string) => api.delete(`/projects/${projectId}/characters/${characterId}`).then((r) => r.data),
  updateLockLevel: (projectId: string, characterId: string, lockLevel: 'loose' | 'medium' | 'strict') => api.put<{ data: Character }>(`/projects/${projectId}/characters/${characterId}/lock-level`, { lockLevel }).then((r) => r.data.data),
};
```

## 前后端交互流程

### 创建角色流程

```
前端 (StudioPage)
    ↓ POST /projects/:projectId/characters
后端 (CharactersController)
    ↓ create()
CharactersService
    ↓ Prisma.create()
数据库 (characters表)
```

### 更新锁定强度流程

```
前端 (锁定强度弹窗)
    ↓ PUT /projects/:projectId/characters/:characterId/lock-level
后端 (CharactersController)
    ↓ updateLockLevel()
CharactersService
    ↓ Prisma.update()
数据库 (更新lockLevel字段)
```

### 提示词生成流程（带锁定强度）

```
生成预览/视频时
    ↓
CharactersService.buildCharacterPrompt(character, lockLevel)
    ↓ 根据lockLevel生成不同详细程度的提示词
传递给ImageAdapter/VideoAdapter
    ↓
调用AI模型生成图像/视频
```
