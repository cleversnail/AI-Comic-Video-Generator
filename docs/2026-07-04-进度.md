## 一、项目当前概况

| 维度 | 状态 |
|------|------|
| 后端框架 | NestJS 10 + Prisma 6.x + MySQL 8.0 |
| 前端框架 | Next.js 14 App Router + Tailwind CSS |
| 数据库 | MySQL（Prisma Schema 已覆盖全部领域模型） |
| 队列系统 | BullMQ + Redis ✅ |
| 对象存储 | MinIO / S3 ✅ |
| 认证 | JWT + bcrypt + svg-captcha ✅ |
| API Key 加密 | AES-256-GCM ✅ |
| AI Adapter | LLM / Image / Video / TTS 按能力拆分 ✅ |
| 已接入模型 | DeepSeek-V3, FLUX, 可灵(图/视频), MiniMax TTS |

**产品定位**：面向普通创作者的 AI 漫剧创作工作台，支持「角色设定 → 剧情输入 → 分镜编排 → AI 视频生成 → 导出」全流程。

## 九、关键成功要素

1. **极速模式优先**：让用户 3 分钟内看到第一个作品
2. **分镜静态预览闭环**：生成视频前先看构图

4. **音频/字幕时间轴前置**：避免音画不同步返工
